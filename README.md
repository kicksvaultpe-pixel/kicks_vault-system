# Kicks Vault — Sistema de Gestión

Sistema de inventario, ventas y flujo de caja para negocio de sneakers y prendas.

## Stack
- Frontend: HTML/CSS/JS (single file)
- Base de datos: Supabase (PostgreSQL)
- Hosting: Netlify

## Setup
1. Ejecutar `supabase-setup.sql` en el SQL Editor de Supabase
2. Abrir `index.html` o acceder via link de Netlify
